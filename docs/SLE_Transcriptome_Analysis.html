Reanalysis of RNA-Seq Data from ‘Multi–Cell Type Gene Coexpression Network Analysis Reveals Coordinated Interferon Response and Cross–Cell Type Correlations in Systemic Lupus Erythematosus’

Systemic lupus erythematosus is a complex autoimmune disease characterized by dysregulated immune responses involving multiple cell types. In this study, we perform a reanalysis of publicly available bulk RNA-seq data from GSE149050 to identify differentially expressed genes and enriched pathways associated with SLE pathogenesis.

7. Functional Enrichment Analysis

To better understand the biological significance of the differentially expressed genes, functional enrichment analysis was performed using the SummArIzeR R package.

# Function to prepare the gene list data frame for one comparison
prepare_genelist_df <- function(res_object, celltype, groupname) {
  
  # Filter for padj < 0.05 and |log2FoldChange| > 1
  filtered <- res_object[!is.na(res_object$padj) & res_object$padj < 0.05 & abs    (res_object$log2FoldChange) > 1, ]
  
  if (nrow(filtered) == 0) {
    return(NULL)
  }
  
  df <- data.frame(
    genes = rownames(filtered),
    Group = groupname,
    CellType = celltype,
    regulation = ifelse(filtered$log2FoldChange > 0, "up-regulated", "down-regulated"),
    log2fold = filtered$log2FoldChange
  )
  
  return(df)
}
